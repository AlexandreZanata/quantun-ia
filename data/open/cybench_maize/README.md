# CY-Bench maize (AgML sample US)

Open dataset wrapper for Research Cycle v2 Phase C / exp_095.

- **License:** EUPL-1.2 (CY-Bench / AgML)
- **Source:** [WUR-AI/sample_data](https://github.com/WUR-AI/sample_data) designed maize US feature tables  
  Full archive: Zenodo DOI [10.5281/zenodo.11502142](https://doi.org/10.5281/zenodo.11502142)
- **Build:** `make data-open-cybench-maize`
- **Dataset ID:** `cybench_maize_us_v1`

Raw CSVs and processed parquet are gitignored; reproduce locally with the make target.
